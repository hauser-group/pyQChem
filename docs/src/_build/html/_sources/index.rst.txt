pyQChem - Input/Output-Tools for Q-Chem
=======================================

PyQChem is a Python module designed for an intuitive manipulation of `Q-Chem <http://www.q-chem.com>`_ input and output files. It was written with special focus on the features of `IPython <http://ipython.org>`_ such as tab completion and easy access to help docstrings via the question mark operator.
