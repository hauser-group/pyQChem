pyqchem.utilities.kabsch(P, Q)¶

The Kabsch algorithm

http://en.wikipedia.org/wiki/Kabsch_algorithm

The algorithm starts with two sets of paired points P and Q. P and Q should already be centered on top of each other.

Each vector set is represented as an NxD matrix, where D is the the dimension of the space.

The algorithm works in three steps: - a translation of P and Q - the computation of a covariance matrix C - computation of the optimal rotation matrix U

The optimal rotation matrix U is then used to rotate P unto Q so the RMSD can be caculated from a straight forward fashion.
